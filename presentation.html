scores_clean <- scores |>
  mutate(
    GameDate_clean = str_remove(GameDate, "(st|nd|rd|th)$"),
    GameDate_full = paste(GameDate_clean, Season),
    GameDate = suppressWarnings(mdy(GameDate_full)),
    total_points = HomeScore + AwayScore,
    point_diff = abs(HomeScore - AwayScore),
    winner = ifelse(HomeScore > AwayScore, HomeTeam, AwayTeam),
    loser = ifelse(HomeScore < AwayScore, HomeTeam, AwayTeam),
    home_win = HomeScore > AwayScore
  )
regular <- scores_clean |>
  filter(GameStatus == "Regular")

In particular, I focused on the date format. I also created new variables for analysis, and make sure to locate the winner and loser of each game.

The total points include the total points scored by both teams, the point difference is the difference in points between the home and away scores. The winner and loser columns include the team that won and lost the game. The home win column expresses whether the home team that won the game.

regular <- scores_clean |>
  filter(GameStatus == "FINAL") |>
  filter(str_detect(Week, "^WEEK"))

season_summary <- regular |>
  group_by(Season) |>
  summarise(
    avg_points = mean(total_points, na.rm = TRUE),
    avg_diff = mean(point_diff, na.rm = TRUE),
    games = n(),
    high_scoring = sum(total_points >= 50, na.rm = TRUE),
    close_games = sum(point_diff <= 3, na.rm = TRUE)
  )


# While using the clean dataset, I created a summary to look at the trends between scoring and game competitiveness over time. In the code, I first filtered the data to include just completed games.I then filtered the dataset again to include the variable week. This would remove and special events or preseason games to order consistency in comparisons. 

# I then grouped the data by season and calculated several summary statistics. The summary statistics include the average points, average difference, number of games, high scoring games, and close games. 

# There were several insights from the data. One of the insights includes the number of games per season. The number of game per season includes 256, and has increased to 272 in 2021. This was when the NFL expanded the regular season schedule. The average scoring per game does fall in the mid-40s, with a peak in 2020 with 49.5 points. The average point difference is around 10 to 12 points, which was relatively stable.

library(ggplot2)

team_season_points <- regular |>
  mutate(
    team = HomeTeam,
    points = HomeScore
  ) |>
  bind_rows(
    regular |>
      mutate(
        team = AwayTeam,
        points = AwayScore
      )
  ) |>
  group_by(team, Season) |>
  summarise(
    avg_points = mean(points, na.rm = TRUE),
    .groups = "drop"
  )


team_heatmap <- team_season_points |>
  ggplot(aes(x = Season, y = team, fill = avg_points)) +
  geom_tile(color = "white") +
  scale_fill_viridis_c(option = "plasma", name = "Avg Points") +
  labs(
    title = "NFL Team Scoring Heatmap",
    x = "Season",
    y = "Team"
  ) + 
  theme_minimal(base_size = 14) +
  theme(
    axis.text.y = element_text(size = 8)
  )
