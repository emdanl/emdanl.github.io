Project Overview

  • Project Webpage: [https://emdanl.github.io]

The project looks into the scoring trends in the National Football League. The dataset includes the years 2010 through 2025. The dataset includes 5,832 games from the regular season, postgames season, and special events. This allows analysis to compare how scoring has changed over time, how team performance varies across seasons, and how competitive games are. This particular project includes descriptive statistics, visualizations, data transformations, along with an interactive dashboard.

Big Idea

The NFL dataset proves that the NFL scoring has been relatively stable over the past 15 years. The average games tend to be scoring in the 40s, and high scoring games are competitive but rare. The team performance tends to vary from season to season, which supports the idea that the league can be unpredictable.

Guiding Question or Story Theme

How have scoring patterns, team performance, and competitiveness, changed across the NFL seasons from 2010 to 2025.

Background and Context

The National Football League is one of the most watched sports leagues in the United States. Scoring changes in the NFL are closely related to rule chagnes, team performances, and offensive strategies. Showing how scoring has changed, and understanding why, provides insights into competitiveness of the league, offensives changes, and heightens the fans experience.

Data Preparation and Transformation

These transformations standardize dates, identify the winning and losing teams, filters out the preason and special events games, and creates more scoring metrics.

scores_clean <- scores |>
  mutate(
    GameDate_clean = str_remove(GameDate, "(st|nd|rd|th)$"),
    GameDate_full = paste(GameDate_clean, Season),
    GameDate = suppressWarnings(mdy(GameDate_full)),
    total_points = HomeScore + AwayScore,
    point_diff = abs(HomeScore - AwayScore),
    winner = ifelse(HomeScore > AwayScore, HomeTeam, AwayTeam),
    loser = ifelse(HomeScore < AwayScore, HomeTeam, AwayTeam),
    home_win = HomeScore > AwayScore
  )
regular <- scores_clean |>
  filter(GameStatus == "Regular") |>
  filter(str_detect(Week, "^WEEK"))

regular <- scores_clean |>
  filter(GameStatus == "FINAL") |>
  filter(str_detect(Week, "^WEEK"))

season_summary <- regular |>
  group_by(Season) |>
  summarise(
    avg_points = mean(total_points, na.rm = TRUE),
    avg_diff = mean(point_diff, na.rm = TRUE),
    games = n(),
    high_scoring = sum(total_points >= 50, na.rm = TRUE),
    close_games = sum(point_diff <= 3, na.rm = TRUE)
  )
